1""" 

2Channel 

3 

4The channel class represents the out-of-character chat-room usable by 

5Accounts in-game. It is mostly overloaded to change its appearance, but 

6channels can be used to implement many different forms of message 

7distribution systems. 

8 

9Note that sending data to channels are handled via the CMD_CHANNEL 

10syscommand (see evennia.syscmds). The sending should normally not need 

11to be modified. 

12 

13""" 

14 

15from evennia.comms.comms import DefaultChannel 

16 

17 

18class Channel(DefaultChannel): 

19 """ 

20 Working methods: 

21 at_channel_creation() - called once, when the channel is created 

22 has_connection(account) - check if the given account listens to this channel 

23 connect(account) - connect account to this channel 

24 disconnect(account) - disconnect account from channel 

25 access(access_obj, access_type='listen', default=False) - check the 

26 access on this channel (default access_type is listen) 

27 delete() - delete this channel 

28 message_transform(msg, emit=False, prefix=True, 

29 sender_strings=None, external=False) - called by 

30 the comm system and triggers the hooks below 

31 msg(msgobj, header=None, senders=None, sender_strings=None, 

32 persistent=None, online=False, emit=False, external=False) - main 

33 send method, builds and sends a new message to channel. 

34 tempmsg(msg, header=None, senders=None) - wrapper for sending non-persistent 

35 messages. 

36 distribute_message(msg, online=False) - send a message to all 

37 connected accounts on channel, optionally sending only 

38 to accounts that are currently online (optimized for very large sends) 

39 

40 Useful hooks: 

41 channel_prefix() - how the channel should be 

42 prefixed when returning to user. Returns a string 

43 format_senders(senders) - should return how to display multiple 

44 senders to a channel 

45 pose_transform(msg, sender_string) - should detect if the 

46 sender is posing, and if so, modify the string 

47 format_external(msg, senders, emit=False) - format messages sent 

48 from outside the game, like from IRC 

49 format_message(msg, emit=False) - format the message body before 

50 displaying it to the user. 'emit' generally means that the 

51 message should not be displayed with the sender's name. 

52 

53 pre_join_channel(joiner) - if returning False, abort join 

54 post_join_channel(joiner) - called right after successful join 

55 pre_leave_channel(leaver) - if returning False, abort leave 

56 post_leave_channel(leaver) - called right after successful leave 

57 pre_send_message(msg) - runs just before a message is sent to channel 

58 post_send_message(msg) - called just after message was sent to channel 

59 

60 """ 

61 

62 pass
